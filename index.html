E. Coli Antibiotic Resistance by Gene Content - Fall 2019 (Present)

Tools Used: Python [BioPython, Pandas, Scipy], NCBI BLAST+, NCBI Database

This project has been awarded the Mulcahy Fellowship at Loyola University Chicago. The project focuses on finding the genes responsible for antibiotic resistance in 77 different strains of E. Coli. We are attempting this by focusing on the variations in gene content between phenotypes that may indicate the reason for the resistance.

Codon Bias Correlation Indicating Phage Host - Summer 2018 (Present)

Tools Used: Python [BioPython, Pandas, Scipy], NCBI BLAST+, NCBI Database

In collaboration with Dr. Catherine Putonti, this project is searching for the bacterial host of many different phage species. We do this by calculating the biases in their codon usage and matching it to different phages which may indicate host status.

LUC Status - Summer 2019 (Present)

Tools Used: Ruby on Rails, Javascript, HTML, CSS, Postgresql

Solving the issues if inconsistent dining halls and printers at Loyola University Chicago. This web application allows students to vote on the status of printers around campus as well as the quantity, and quality, found at the Dining halls. Students can access this portal to view the information.

Housing Price Prediction Kaggle Competition - Spring 2019

Tools Used: Python, Jupyter Notebooks, Scipy, Scikit learn, Pandas

As the final project for my Data Structures course, my group entered the Kaggle competition for predicting the prices of houses. We learned machine learning principles and were able to predict house prices within the +/- $6k range.

Spa Customer Management Software - Summer 2018

Tools Used: PHP, HTML, CSS, Javascript [jQuery], MySQL

Built for the SilksUSA Salon & Spa. This application is hosted on a on premises Raspberry Pi. Employees at the spa are now able to onboard customer consent forms entirely online in addition to collecting email addresses and phone numbers for marketing reasons. This is improved from the old pen/paper system.

Asteroids Game (Android App Store) - Spring 2018

Tools Used: Unity Game Engine, C#, GIMP, Audacity

As a larger project in game development I created a remake of the classic Asteroids game. This remake was created in the Unity Game Engine and coded in C#. The app is currently available in the Google Play Store and has 200+ installs.

Two Three Five Game - Spring 2018

Tools Used: Java, IntelliJ IDE

As a final project for my Intro to OOP course. I remade an Indian card game in the Java console. This allows players to replay the game either against another human player or against a bot programmed to play along with the correct rules and understanding as any other human.

Hangman Game - Fall 2017

Tools Used: Java, IntelliJ IDE

To introduce OOP principles, this project was made to better understand the IntelliJ IDE. This was a useful beginner project for learning how to make simple applications in Java.
